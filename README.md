## Технологии

- Аналог virtual-dom для парсинга html шаблонов
- Класс BaseComponent для работы с компонентами
- Класс EventBus для реализации жизненного цикла компонента
- Валидация форм с помощью регулярных выражений
- Typescript
- ESLint
- Stylelint
